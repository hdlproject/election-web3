# chain-sdk (Minimal)

Lightweight helpers to interact with your smart contracts using ethers + ABIs. No Alchemy, no extras.

## Prerequisites
Generate ABIs in backend:
```bash
cd backend
yarn run abi
```
Sync ABIs into SDK:
```bash
cd chain-sdk
yarn run sync:abi
```

## Install & Build
```bash
cd chain-sdk
yarn install
yarn run build
```
